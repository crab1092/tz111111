# HTTP Client & Server для сбора информации из интернета

## Описание проекта

Проект представляет собой HTTP-сервер, реализованный на платформе Node.js с использованием фреймворка Express.

Сервер хранит список ключевых слов и соответствующих им интернет-ресурсов. Клиент может отправить ключевое слово на сервер, получить список связанных URL-адресов, а затем запросить загрузку содержимого выбранной страницы.

Проект демонстрирует работу с HTTP-запросами, REST API и взаимодействие клиент-серверных приложений.

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/crab1092/tz111111.git
```

### 2. Переход в папку проекта

```bash
cd tz111111
```

### 3. Установка зависимостей

```bash
npm install
```

### 4. Запуск сервера

Обычный режим:

```bash
npm start
```

Режим разработки:

```bash
npm run dev
```

После запуска сервер будет доступен по адресу:

```
http://localhost:3000
```

В консоли появится сообщение:

```
Server started on port 3000
```

## API сервера

### Проверка работы сервера

**GET /**

Возвращает сообщение о том, что сервер запущен.

Пример ответа:

```json
Server is working
```

---

### Получение списка URL по ключевому слову

**POST /search**

Тело запроса:

```json
{
  "keyword": "javascript"
}
```

Пример ответа:

```json
{
  "urls": [
    "https://javascript.info",
    "https://developer.mozilla.org/en-US/docs/Web/JavaScript"
  ]
}
```

---

### Загрузка содержимого веб-страницы

**POST /download**

Тело запроса:

```json
{
  "url": "https://javascript.info"
}
```

Пример ответа:

```json
{
  "content": "...содержимое страницы...",
  "size": 12345
}
```

---

## Особенности реализации

* Обработка HTTP GET и POST запросов.
* Хранение данных в отдельном JavaScript-модуле.
* Работа с внешними веб-ресурсами через Axios.
* Обработка ошибок при некорректных запросах и недоступности сайтов.
* Поддержка обмена данными в формате JSON.
